# 前端管理界面

基于 Vue 3 + Element Plus 的电商自动化管理后台

## 功能模块

- 📊 数据看板 - 实时运营数据
- 📦 订单管理 - 订单列表、详情、处理
- 🛍️ 商品管理 - 商品列表、库存、价格
- 🤖 智能客服 - 对话记录、知识库
- 💰 价格监控 - 竞品价格、调价建议
- 📈 数据报表 - 销售分析、趋势图表
- ⚙️ 系统设置 - 配置管理、权限控制

## 技术栈

- Vue 3 + TypeScript
- Element Plus UI
- ECharts 图表
- Axios HTTP 客户端
- Pinia 状态管理

## 快速开始

```bash
cd frontend
npm install
npm run dev
```

## 构建部署

```bash
npm run build
```

构建后的文件在 `dist/` 目录，可部署到 Nginx 或 CDN。
